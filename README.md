# Верстка лендинга

## Основной функционал:
- Плавная навигация по сайту
- Оптимизирована скорость загрузки
- Отправка данных с формы на почту
- Анимация с помощью animate.css
- Сайт отлично отображается на любом устройстве (кроссбраузерность)
- ## Инструменты 
- Gulp, SCSS
